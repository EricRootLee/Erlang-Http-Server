HTPP SERVER
=====

A Simple Erlang http server with just 2 get routes/endpoints

Run
-----

    $ rebar3 shell
    localhost:800/simple
    localhost:8080/add/87/98

Build
-----

    $ rebar3 compile

Dependances
------

   Cowboy
